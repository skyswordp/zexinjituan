# 流程图使用说明

## 方式1：使用Excalidraw扩展（推荐）

1. **安装扩展**
   - 按 `Ctrl+Shift+X` 打开扩展面板
   - 搜索 `Excalidraw`
   - 安装 `pomdtr.excalidraw-editor`

2. **打开流程图**
   - 双击 `朋友圈交互流程图.excalidraw` 文件
   - 即可看到手绘风格的精美流程图
   - 支持自由编辑、拖拽、美化

3. **特点**
   - ✨ 手绘风格，比Mermaid好看100倍
   - 🎨 支持多种颜色和样式
   - 📝 可以添加emoji和图标
   - 🖱️ 鼠标拖拽即可编辑

## 方式2：在线编辑

如果扩展安装失败，可以：

1. 访问 https://excalidraw.com/
2. 点击 `文件` → `打开`
3. 选择 `朋友圈交互流程图.excalidraw` 文件
4. 在线编辑后保存

## 方式3：导出图片

1. 用Excalidraw打开文件
2. 点击 `导出` → `PNG` 或 `SVG`
3. 保存为图片插入到其他文档中
